Loops:

Loops are sequences of instruction that are repeated until a certain condition is reached. The two types of loops are for loops and while loops. For loops repeat a sequence for a known number of times. While loops repeat a sequence for a unknown number of times.

Python ex.: 

while lives > 0:
                            run()

Conditionals:

Conditionals are if/then statements that evaluate a boolean condition as true or false then performs a sequence. If a conditional requres multiple conditions, then an if/else loop can be used. If the first condition is not met, then a different sequence is performed.

Python ex.: 

if lives = 0:
                            left(180)
                         elif lives > 0:
                            run()
